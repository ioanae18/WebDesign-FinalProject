    Fierbator Star-Light DK 382, 2200 W, 1.7 l, Negru/Inox

    Fie ca va doriti un ceai in cel mai scurt timp posibil sau sa realizati un adevarat ritual al ceaiului, fierbatorul Star-Light Kettle DK-382 va ofera posibilitatea prepararii unei bauturi calde rapid si in siguranta. Desi urmeaza un design modern, inox-ul de cea mai buna calitate ii ofera un aspect deosebit, ce aduce aminte de ibricele clasice.

    Avand o cantitate de 1.7 l si o putere de 2200 W fierbatorul Star-Light Kettle DK-382 este optiunea ideala pentru a incalzi apa rapid si eficient. Display-ul gradat va ofera posibilitatea de a sti ce cantitate de apa incalziti.

    Fierbatorul Star-Light Kettle DK-382 este prevazut cu un sistem de iluminare intern, care indica atunci cand este pornit. De asemenea, sistemul de pornire permite oprirea circuitului de fiecare data cand ibricul este detasat.

    Capacul fierbatorului Star-Light Kettle DK-382 este dotat cu un buton de siguranta, care permite deschiderea facila. Palnia este prevazuta cu un sistem de filtrare demontabil impotriva reziduurilor si a depunerilor de calcar.

    Baza rotativa de 360ยบ permite pozitionarea ibricului din orice directie, astfel se poate ridica facil atat cu mana dreapta, cat si cu mana stanga. De asemenea, fierbatorul este prevazut cu un sistem de siguranta, care impiedica pornirea atunci cand nu este apa in interior sau cand apa este deja la temperatura potrivita.
